## Organization

**Minimal configuration**:

```txt
./
├── ...
├── main.py
├── offline
│   ├── ...
│   └── index.html
├── theme
│   ├── ...
│   └── theme.css
└── web
    ├── ...
    ├── index.html
    └── logic.js
```

## Theme and color scheme

- `theme/theme.css`: Style sheet for the search engine theme
